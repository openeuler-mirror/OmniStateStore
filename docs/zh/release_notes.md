## V1.1.0<a name="ZH-CN_TOPIC_0000002548934227"></a>

### 更新说明<a name="ZH-CN_TOPIC_0000002517334438"></a>

当前版本旨在解决大数据场景下，针对大状态下IO性能较差的问题，实现了一种新型的状态存储方式，提升了Flink的IO性能。

**新增特性<a name="section9559165534817"></a>**

- 支持对接Flink Metric框架，并实现部分常用的Metric指标。
- 支持Priority Queue持久化存储。
- 支持KV分离存储。
